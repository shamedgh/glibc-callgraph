19 /* The PLT stubs will call _dl_runtime_resolve/_dl_runtime_profile
20  * with the following linkage:
21  * r2 - r6 : parameter registers
22  * f0, f2, f4, f6 : floating point parameter registers
23  * v24, v26, v28, v30, v25, v27, v29, v31 : vector parameter registers
24  * 48(r15), 56(r15) : PLT arguments PLT1, PLT2
25  * 160(r15) : additional stack parameters
26  * The normal clobber rules for function calls apply:
27  * r0 - r5 : call clobbered
28  * r6 - r13 : call saved
29  * r14 : return address (call clobbered)
30  * r15 : stack pointer (call saved)
31  * f0 - f7 : call clobbered
32  * f8 - f15 : call saved
33  * v0 - v7 : bytes 0-7 overlap with f0-f7: call clobbered
34  bytes 8-15: call clobbered
35  * v8 - v15 : bytes 0-7 overlap with f8-f15: call saved
36  bytes 8-15: call clobbered
37  * v16 - v31 : call clobbered
38  */
